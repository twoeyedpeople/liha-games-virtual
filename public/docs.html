LiHa Games Guide

How to run sessions from a people manager perspective.

Back to Aux Open Control Open Stats

1. Venue Setup

  • Open /aux on your laptop for quick links and previews.
  • Assign displays: Intro + Leaderboard on landscape TVs, Trust Who + Soft Skills Significance Countdown on portrait TVs.
  • Open leaderboard in single mode and use Claim Leaderboard in control to switch the display between Group A and Group B.
  • Open /control.html on a manager-only device.
  • Check session A and B state before admitting players.

2. Trust Who (Game 1)

  • Start Trust Who in control for each session.
  • Mark a team finished when they complete the challenge.
  • The countdown display reveals final times and Trust Who points.
  • Those points flow into combined leaderboard totals after reveal.

3. Soft Skills Significance (Game 2)

  • Soft Skills Significance runs as 3 rounds, each 2 minutes.
  • Start each new round manually from control.
  • Mark a team finished when they complete the challenge.
  • The display shows the active round and timer for that round only.
  • Round points accumulate across all 3 rounds and then flow into leaderboard totals.

4. Prompt Like a Pro (Game 3)

  • Open lobby for the session and share passcode.
  • Players pick team, role brief, then submit one prompt.
  • Prompt scoring uses role criteria (Role details, Context, Responsibilities).
  • At round end, game points are revealed first, then merged into leaderboard.

5. Recruiter Focus Finder (Intro)

  • Collect company name and task selections.
  • The intro estimates weekly AI time savings from unselected automatable tasks.
  • Selections and savings are stored in stats for post-event analysis.

6. During the Event

  • Use pause/resume controls for interruptions.
  • Use reset only when a session must be fully wiped.
  • If network drops, refresh control first, then external displays.

7. After the Event

  • Open /stats to review aggregate performance.
  • Download XLS for offline reporting or stakeholder share-out.
  • Reset sessions in control before the next cohort starts.
